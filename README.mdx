# recipe

S64/recipe

## 目的

下記の達成を助ける為の文書を作る

-   主に [@S64](https://github.com/S64) が異なる環境でも可能な限り再現性高く料理を作れる
-   可能な限り火を用いずに料理を作れる
-   可能な限り少ない調理器具の種類で料理を作れる

## 方針

### 改修

-   デフォルトブランチを `snapshot` とし、既知の問題がある場合にはそれを読み取れる形で記す
-   各レシピは `/posts` 内に `^[a-z0-9-_]+\.mdx$` のファイル名で配置する
-   改修は Visual Studio Code を用いることを想定し、拡張機能等の設定もこれで利用できるものにする

### 買い物

-   大手小売店で購入可能な食材を原則とし、調達難易度を下げる
-   そのコストが十分に許容可能な限り、カット済み野菜等の下ごしらえが完了している食材を用いる

### 使用器具

-   平皿、椀など標準的な皿である限り、レシピには記載しない
-   [シャープ ヘルシオ ホットクック](https://jp.sharp/hotcook/) については、下記を利用できるものとする
    -   `KN-HW10G`
    -   `KN-HW16G`
